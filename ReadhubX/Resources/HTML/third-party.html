Readhub X 第三方内容条款

Readhub X 是一款无码科技官方产品 Readhub 的第三方客户端。

Readhub X 所提供的内容和服务均归无码科技（杭州）有限公司所有。如您浏览访问使用 Readhub X App，即表示您自愿接受本服务协议的全部条款以及内容，并同意接受由无码科技（杭州）有限公司为您提供的服务。

当您选择【开始使用】，即表示接受为您推荐来自第三方网站的内容，主要展示的内容包括：热门话题、话题详情、科技动态、开发者资讯和区块链快讯。

如您对本服务协议的任何内容存在异议，请立即停止使用 Readhub X 的内容服务。请您详情阅读本协议的全部内容。


Readhub X 隐私策略

我们尊重并保护您的个人隐私。本隐私策略解释 Readhub X 收集的关于您的哪些信息及其原因，以及会如何使用和保护这些信息。

Readhub X 会收集我的哪些信息

Readhub X 以匿名形式通过 Bugly 服务收集日常使用数据、崩溃日志。

Readhub X 如何保护我的隐私

Readhub X 不会在除 Bugly 以外的服务器存储数据。

Readhub X 不会在除非您许可的情况下将您的个人信息分享给第三方。
